The scale-probe evidence is `governed-memory-probe.json`. It is generated by:

```bash
python3 paper/scripts/run_governed_memory_probe.py \
  --records 10000 \
  --iterations 25 \
  --candidate-cap 200 \
  --output paper/data/governed-memory-probe.json
```

support the narrow mechanism described in the paper; the latency fields are
not portable performance guarantees. The probe uses in-memory SQLite and does
not include durable I/O or macOS sealing cost.
